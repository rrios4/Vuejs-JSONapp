How to Run
----------------------------------------------------------
Run this command in client and root folder to install dependencies:
npm install

To start the application run:
npm run dev

Go to this link to see Vue page:
http://localhost:8080/

To view the JSON data go to:
http://localhost:5000/api/persons/
